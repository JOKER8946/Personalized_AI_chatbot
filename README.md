# Personalized AI Chatbot

A modern, responsive AI chatbot that gets to know you! Customize your bot's personality and see it adapt its responses to your interests and name.

## ✨ Features
- **Personalization:** Set your name, hobbies, and preferred bot tone in the sidebar.
- **Gemini AI Integration:** Powered by Google's `gemini-1.5-flash` model for fast and intelligent responses.
- **Persistence:** Your profile and chat history are saved automatically in your browser's local storage.
- **Modern UI:** Built with Tailwind CSS and custom animations for a smooth experience.

## 🚀 Getting Started
1. Open `index.html` in your browser.
2. You will be prompted to enter your **Gemini API Key**.
   - If you don't have one, get it for free at [Google AI Studio](https://aistudio.google.com/app/apikey).
3. Update your preferences in the sidebar and click **Save Preferences**.
4. Start chatting!

## 🛠️ Built With
- HTML5 / Tailwind CSS
- Vanilla JavaScript
- Google Gemini API
